## **STL 컨테이너 (STL Container)**

1. 순차 컨테이너 (Sequence Container)

    + vector

    ```c++
    #include<iostream>
    using namespace std;
    ```

    ```c++
    // 생성 및 초기화

    vector<자료형> v(size, value);
    // default size & value = 0
    // Parameter로 vector를 전달하면 복사 가능

    v.assign(size, value)

    ```

    ```c++
    // 원소 참조

    v.at(index)
    v[index]

    // at은 범위를 점검 하므로 안전
    // []은 범위를 점검하지 않아 빠름
    ```

    + deque

    ```c++
    #include <deque>
    using namespace std;
    ```

    + list

    ```c++
    #include <list>
    using namespace std;
    ```
